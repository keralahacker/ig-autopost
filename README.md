# ig-autopost
Automation script for Instagram photo upload 


## Step 1:
   Install ‘instabot‘ package using the below command in the terminal. 
```
pip install instabot
```

## Step 2 :
```
   chmod +x *
```

## Step 3 :
```
   python ig-post.py
```
